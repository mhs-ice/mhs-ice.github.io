1. git init (in that folder where is your project)
2. git remote add origin https://github.com/mhs-ice/MyRepo.git
3. git add .
4. git commit -m "Initial"

If it is the initial commit then 
5. git branch -M main
6. git push -u origin main

Otherwise except for the initial commit

1. git add .
2. git commit -m "......"
